\# 🌟 \*\*Queens Of Change Foundation - Website Project\*\*



A modern, responsive NGO landing page built using \*\*HTML and CSS\*\* for an internship selection task.



This project focuses on \*\*clean UI design, responsiveness, and meaningful content presentation\*\*.



\---



\## 🎯 \*\*Objective\*\*



\- Create a simple yet professional NGO website  

\- Demonstrate \*\*HTML \& CSS skills\*\*  

\- Build a \*\*responsive and user-friendly layout\*\*  

\- Show creativity in \*\*UI/UX design\*\*



\---



\## ✨ \*\*Features\*\*



✔ \*\*Modern hero section\*\* with call-to-action button  

✔ Clean and structured layout  

✔ About section with meaningful description  

✔ Mission cards (Education, Community, Opportunity)  

✔ Image section to showcase impact  

✔ Volunteer / Join Us section  

✔ Fully responsive design (mobile + desktop)  

✔ Smooth and minimal UI styling  

✔ Hover effects for better interaction  



\---



\## 🛠️ \*\*Tech Stack\*\*



\- \*\*HTML5\*\*  

\- \*\*CSS3\*\*  

\- \*\*Flexbox for layout\*\*  

\- \*\*Responsive Design principles\*\*  

\- \*\*Unsplash images\*\*
